# rea

## Description

My blog URL: https://aiktb.com

Personal blog based on [VitePress](https://vitepress.dev/) default theme extension.

Q: How to develop?

A: Read this [blog](https://aiktb.com/posts/vitepress-blog)(Chinese).

- Simple tones
- Dark mode only
- Recent blogs
- Google Analytics support
- RSS feed
- Local search
- Comments
